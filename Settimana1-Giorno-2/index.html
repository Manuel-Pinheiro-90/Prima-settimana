Formattazione codice


il testo dell'articolo dovrà esser formattato per identificare tutte le parti da selezionare con css. Perciò i passi saranno:

  1. progettare murkup assegnando ad ogni testo il tag appropriato
  2. scrivere html dai tag esterni a quelli interni dall'alto verso il basso
  3. applicare gli stili CSS
